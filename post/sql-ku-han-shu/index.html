SQL库函数

2020-08-25 1 min read # sql

rank函数

  1. rank()       是跳跃拍学，两个第二名下来就是第四名
  2. dense_rank() 是连续排序，两个第二名仍然跟着第三名

语法：

Select RANK() OVER ( 
    [PARTITION BY partition_expression, ... ] 
    ORDER BY sort_expression [ASC | DESC], ...
) from Table
  • PARTITION BY子句划分应用该函数的结果集分区的行。
  • ORDER BY子句指定应用该函数每个分区中行的逻辑排序顺序。

round 函数

用于把数值字段舍入为指定的小数位数

  • 语法：
    SELECT ROUND(column_name,decimals) FROM table_name

datediff(A,B)

日期A和B之间差的天数

      • rank函数
      • round 函数
