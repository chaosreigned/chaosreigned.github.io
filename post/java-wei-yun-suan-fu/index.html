java位运算符

2020-08-26 1 min read

移位

<< : 左移运算符，num << 1,相当于num乘以2
>> : 右移运算符，num >> 1,相当于num除以2
>>> : 无符号右移，符号位都以0补齐

位运算

位与：&
位或：|
位异或：^

x ^ 0s = x x & 0s = 0 x | 0s = x
x ^ 1s = ~x x & 1s = x x | 1s = 1s
x ^ x = 0 x & x = x x | x = x

x & (-x) 是保留位中最右边 1
