牛顿迭代法计算开方

2020-08-26 1 min read

x^2 = n;
f(x) = x^2 -n;
f(x)上每个点点切线斜率都为2x，因此f(x)的切线和x轴的交点为：x-f(x)/2x = (x+n/x)/2;
从x为n开始，求得f(x)上该点切线与x轴的交点，会慢慢逼近最终达到f(x)和x轴的交点，及所求的开方值

代码：

int n;
public double sqrts(double x){
    double res = (x + n/ x) / 2;
    if (res == x) {
        return x;
    } else {
        return sqrts(res);
    }
}
